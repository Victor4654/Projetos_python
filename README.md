## 🚀 Como executar

**Pré-requisitos:** Python 3.x instalado na máquina.

1. Clone o repositório:
```bash
git clone https://github.com/Victor4654/Projetos_python.git
```

2. Acesse a pasta do projeto desejado:
```bash
cd Projetos_python/Conversor_de_unidades
```

3. Execute o script:
```bash
python conversor.py
```
